# 3-Input OR Gate using Transistors

## Project Overview

This project demonstrates the design and simulation of a **3-Input OR Logic Gate using NPN Transistors**, developed as part of Basic Electronics coursework.

The circuit implements OR logic functionality using transistor switching principles and validates digital logic behavior through simulation in Tinkercad.

---

## Objective

To design and test a transistor-based 3-input OR gate and analyze logic behavior for all possible input combinations.

---

## Working Principle

The circuit produces HIGH output when:

- Input A is HIGH
- Input B is HIGH
- Input C is HIGH
- Any combination of inputs is HIGH

The output remains LOW only when all three inputs are LOW.

---

| Input A | Input B | Input C | Output |
